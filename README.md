# Extended Survival Shooter

## Deskripsi Aplikasi

Extended Survival Shooter adalah single-player story based game yang merupakan ekstensi dari Survival Shooter klasik.

## Library yang Digunakan

Unity versi 2021.3.21f1 LTS
